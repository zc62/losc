Convert density fitting matrix three-body integral <p|mn> into <p|ii> matrix block by block.

i, j: LO index.
p, q: fitbasis index.
m, n: AO basis index.
The <p|mn> matrix has dimension of [nfitbasis, nbasis * (nbasis + 1) / 2]. The <p|mn> integral is associated with one fitbasis and two AO basis. It is defined as

\[ \langle \phi_p | \phi_m \phi_n \rangle = \int \frac{\phi_p(\rm{r}) \phi_m(\rm{r'}) \phi_n(\rm{r'})}{|\rm{r} - \rm{r'}|} \mbox{d} \rm{r} \mbox{d} \rm{r'}, \]

The <p|ii> matrix has dimension of [nfitbasis, nlo]. The <p|ii> integral is associated with one fitbasis and one LO, and it is defined as

\[ \langle \phi_p | \phi_i \phi_i \rangle = \int \frac{\phi_p(\rm{r}) \phi_i(\rm{r'}) \phi_i(\rm{r'})}{|\rm{r} - \rm{r'}|} \mbox{d} \rm{r} \mbox{d} \rm{r'}, \]

This function convert a block (block of fitbasis indices) of <p|mn> matrix into <p|ii> matrix. The format of <p|mn> block is required and illustrated as the following code.

for (size_t p = 0; p < block_size; ++p) {
for (size_t m = 0; m < nbasis; ++m) {
for (size_t n = 0; n <= m; ++n) {
// set each element <p|mn> in the block as zero.
df_pmn_block(p, m * (m + 1)/2 + n) = 0;
}
}
}
Parameters
[in]p_indexthe corresponding fitbasis index in df_pmn_block. For example, the i-th row in df_pmn_block matrix coresponding to p_index[i]-th fitbasis.
[in]df_pmn_blockdensity fitting <p|mn> matrix block.
[in]C_loLOs coefficient matrix under AOs with dimension of [nbasis, nlo]. The i-th column in C_lo matrix is the i-th LO.
[in,out]df_piidensity fitting <p|ii> matrix. On exit, the rows whose index store in p_index vector are updated.
Note
To build the whole <p|ii> matrix, you need to traverse all the <p|mn> blocks.
