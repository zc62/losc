Detailed Description

LOSC localization version 2.

Note
This is the version used in the LOSC2 paper.
See also
J. Phys. Chem. Lett. 2020, 11, 4, 1528–1535.

double losc::LocalizerV2::cost_func ( ConstRefMat &  C_lo) const
overridevirtual

Return the relative cost function of LOSC localization v2.

Parameters
[in]C_loLOs coefficient matrix under AOs with dimension of [nbasis, nlo]. The i-th column in C_lo matrix is the i-th LO.
Note
The cost function is defined as:

\[ F = \sum_i (1 - \gamma) \Big ( \langle i | \mathbf{r}^2| i \rangle - \langle i | \mathbf{r} | i \rangle ^2 \Big) + \gamma C \Big ( \langle i | H^2| i \rangle - \langle i | H | i \rangle ^2 \Big), \]

in which index i is summed over all LOs. Since the unitary transformation does not change the trace of a matrix, The relative cost function is evaluated:

\[ F_{\rm{rel}} = \sum_i (1 - \gamma) \langle i | \mathbf{r} | i \rangle ^2 + \gamma C \langle i | H | i \rangle ^2. \]

Calculate the LOs and the unitary transformation matrix from the LOSC localization v2.

Parameters
[in]U_guessthe initial guess of U matrix. Its data will be copied. Its unitarity will be verified and throw an exception if the validation fails.
[in]thresholdthe threshold used to check the unitarity. Default to 1e-8.
Returns
a vector of matrix rst with size of 2. rst[0] is the LO coefficient matrix. rst[1] is the corresponding U matrix. The relation between the LOs and AOs is \( \displaystyle \psi_i = \sum_\mu C_{\mu i} \phi_{\mu}, \) in which \( \psi_i \) is the LO, \( C_{\mu i} \) is the LO coefficient matrix and \( \phi_i \) is the AO. The dimension of LO coefficient matrix is [nbasis, nlo]. The relation between the LOs and LO basis is \( \displaystyle \psi_i = \sum_j U_{\mu i} \phi_\mu, \) in which \( \psi_i \) is the LO, \( U_{\mu i} \) is the U matrix and \( \phi_\mu \) is the LO basis. The dimension of the U matrix is [nlo, nlo].
